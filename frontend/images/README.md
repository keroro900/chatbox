# Keroro 作战室图片目录

请将以下图片文件放置在此目录中：

## 需要的图片文件

### 主要LOGO和头像
1. **keroro-logo.png** - Keroro 正面头像（用于顶部LOGO）
2. **keroro-salute.png** - Keroro 敬礼姿势（用于API控制室模态框）
3. **keroro-shy.png** - Keroro 害羞/戴口罩版本（用于工作流提示）
4. **keroro-baby.png** - Keroro 婴儿风格/戴眼镜版本（用于任务详情模态框）
5. **keroro-angry.png** - Keroro 生气/新手标志版本（用于API状态卡片）
6. **keroro-pirate.png** - Keroro 海盗装版本（可选，用于其他位置）

### 角色小队成员（用于右侧角色贴纸）
7. **keroro.png** - Keroro 军曹（绿色）
8. **tamama.png** - Tamama 二等兵（黄色）
9. **giroro.png** - Giroro 下士（红色）
10. **kururu.png** - Kururu 曹长（黄色/橙色）
11. **dororo.png** - Dororo 兵长（蓝色）

## 图片要求

- 格式：PNG（支持透明背景）
- 尺寸：建议 200x200 像素或更大（正方形）
- 背景：透明或白色
- 内容：角色的正面头像或全身像

## 使用说明

- **keroro-logo.png** 用于顶部LOGO（最重要）
- 其他keroro变体用于不同的模态框和提示区域
- 角色小队成员图片用于右侧浮动角色贴纸
- 如果图片文件不存在，系统会自动回退到文字/emoji显示

## 图片来源

你可以从以下来源获取图片：
- 官方 Keroro 军曹动画截图
- 官方角色设定图
- 同人作品（需注意版权）

将图片重命名为对应的文件名后放入此目录即可。
